# Trilha — App de 30 dias

Este é o projeto completo do seu app, pronto para publicar.

## Passo a passo pra colocar no ar (grátis)

### 1. Crie uma conta no GitHub
- Acesse https://github.com e crie uma conta grátis (se ainda não tiver)

### 2. Crie um repositório novo
- No GitHub, clique em "New repository"
- Dê um nome, por exemplo `trilha-app`
- Deixe como público ou privado, tudo bem os dois
- Clique em "Create repository"

### 3. Suba os arquivos
- Na página do repositório recém-criado, clique em "uploading an existing file"
- Arraste TODOS os arquivos e pastas deste projeto (mantendo a pasta `src` como pasta)
- Clique em "Commit changes"

### 4. Publique na Vercel
- Acesse https://vercel.com e crie uma conta (pode entrar direto com sua conta do GitHub)
- Clique em "Add New Project"
- Escolha o repositório `trilha-app` que você acabou de criar
- Deixe as configurações padrão (a Vercel já reconhece projetos Vite/React sozinha)
- Clique em "Deploy"

### 5. Pronto!
- Em 1-2 minutos a Vercel te dá um link, tipo `trilha-app.vercel.app`
- Esse é o link que você pode abrir de qualquer lugar, ou colar no Notion como embed

## Sobre o progresso salvo
O app salva o progresso (XP, sequência, temas concluídos) no próprio navegador de quem acessa (localStorage). Isso significa:
- Funciona sem precisar de conta ou login
- O progresso fica salvo naquele navegador/aparelho específico
- Se abrir em outro celular ou computador, o progresso começa do zero nesse outro aparelho
